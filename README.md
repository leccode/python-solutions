# Steps to follow

> - python -m venv {folder_name} => Creates a virtual env to store the deps inside the downloaded repo
> - {folder_name}\Scripts\activate => Inside the repo type the folder name you gave, and run this to activate the env
> - pip install -r requirements.txt => Installs all the needed deps for this repo

# Solutions

- Chapter 1: > python hello_world.py => Inside the chapter1 folder, run the provided command
- Chapter 2: > python queue.py AND python student.py => Inside the chapter2 folder, run the provided command for every python solution. A clear console module is used here to clean the space

- Note: Feel free to experiment with all of these and make your own programs!
